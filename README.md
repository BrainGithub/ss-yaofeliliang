## 支持系统
CentOS 6+

Debian 7+

Ubuntu 12+

## 使用教程
一键搭建ss/ssr：


``` bash
git clone https://github.com/yaofeiliang/ss-yaofeliliang.git

cd  ss-yaofeiliang

./ss-yaofeiliang.sh -i password 1024
```
